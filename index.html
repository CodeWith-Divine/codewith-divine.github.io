Frequently Asked Questions

Find answers to common questions about our projects and purchasing process.

How does the "Buy Me a Coffee" work?

You can choose any price you want for each project. After clicking "Get via Lemon Squeeze," you'll be redirected to a checkout page where you can set your own price. There's a minimum of $3 to cover processing fees, but you can pay more if you find value in the project. Think of it as buying me a coffee for the time and effort I put into creating these projects!

What's included in each project download?

Each project includes: Complete source code (HTML, CSS, JavaScript), detailed documentation with setup instructions, comments explaining key code sections, assets (images, icons), and a responsive design that works on all devices.

Can I use these projects in my portfolio or for commercial use?

Absolutely! You're encouraged to use these projects in your portfolio to showcase your skills. For commercial use, you have full rights to modify and use the code in your projects. However, you cannot resell or redistribute the original source code as a product.

Is technical support included?

Basic questions via email: Yes (within reason)
Debugging help: Yes (for genuine bugs)
Custom development: YES! contact me here for your custom frontend projects.
Teaching/tutoring: No (watch the YouTube tutorial)

Will this code become outdated?

These projects are built with vanilla HTML, CSS, and JavaScript - stable web standards that have been around for 10+ years. They don't require updates because:

  1. No frameworks (React/Vue) that break with updates
  2. No npm packages with security vulnerabilities
  3. No build tools that need maintenance
  4. Browsers are backwards compatible

The code you purchase today will work for 5-10+ years minimum without any changes.

Think of it like buying a bicycle vs. a car:

  1. Car (frameworks) = needs regular maintenance
  2. Bicycle (vanilla code) = just works, forever

How do I get access after purchase?

Immediately after completing your purchase through Lemon Squeeze, you'll receive an email with a download link. You'll also have access to a customer portal where you can download the files anytime. The download includes a ZIP file with all project files organized and ready to use.

What if I need help or have issues with a project?

Each project includes detailed documentation to help you get started. If you encounter any issues, you can email support at codewithdivine@gmail.com. We typically respond within 24 hours and are committed to helping you succeed with our projects.

Do you offer refunds?

Due to the digital nature of our products and the "buy me a coffee" model, we don't offer refunds. However, if you're unsatisfied with a project for any reason, please contact us and we'll work with you to resolve any issues or provide additional support.
